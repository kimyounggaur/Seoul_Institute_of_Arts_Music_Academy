# 서울예대실용음악학원 소개 사이트

서울 양천구 신정동의 서울예대실용음악학원을 소개하는 모바일 우선 반응형 웹사이트입니다.

## 구성

- 보컬, 기타, 베이스, 드럼, 클래식 피아노, 신디사이저, 교회반주 과정 안내
- 학원 제공 연출 이미지 갤러리
- 학원 홍보 음원 4곡 플레이어
- 오시는 길과 공식 채널 연결
- 서버 전송 없이 상담 내용을 정리하는 상담 메모 기능

## 로컬 실행

```bash
npm install
npm run dev
```

품질 확인은 `npm test`와 `npm run lint`로 실행합니다.
